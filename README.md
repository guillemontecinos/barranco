# Barranco

## About
Barranco is a Chilean based music project which existed between 2013 and 2015.
Barranco was composed by:
* [Aarón Montoya-Moraga](http://montoyamoraga.io/)
* [Guillermo Montecinos](http://guillemontecinos.cl/)
* Javier Bermúdez
* Nicolás Peña
* Sebastin Santelices
